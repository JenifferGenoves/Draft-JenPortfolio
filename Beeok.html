UX Challenge - Beeok

Información del Proyecto

  • Categoria: Diseño Plataforma Web
  • Cliente: Beeok
  • Fecha del Proyecto: Julio, 2020
  • Proyecto Demo Desktop:

El desafío en esta oportunidad es mejorar la experiencia de usuario en la plataforma de software de Beeok, realizar una propuesta que sea más amigable con los usuarios y lograr que tengan una mejor experiencia a la hora de navegar en la plataforma. Para esto se considero a todos los actores participantes, enfocándonos en los profesionales del area de prevencion de riesgo e ingeniería en medio ambiente.
La propuesta inicial para la plataforma, considera mejorar aspectos visuales y la usabilidad de esta.
Para lograr que el usuario tenga una mejor experiencia en la plataforma consideremos mejorar aspectos como gráficos y tablas resumen, para que la información sea mejor visualizada, así como mejorar el área de las notificaciones para que el usuario esté al tanto de las últimas leyes que se van actualizando.
